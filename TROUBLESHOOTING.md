# Решение проблем с запуском

## Проблема: Metro Bundler застревает на "Waiting on http://localhost:8081"

### Решение 1: Отключите антивирус временно
Metro Bundler может блокироваться антивирусом Windows или файерволом.

1. Временно отключите антивирус
2. Запустите: `npx expo start -c`

### Решение 2: Запуск в режиме туннеля
```bash
npx expo start --tunnel
```

### Решение 3: Запуск с другим портом
```bash
npx expo start --port 19000
```

### Решение 4: Запуск веб-версии для тестирования
```bash
npx expo start --web
```

### Решение 5: Очистка всех кэшей
```bash
npm cache clean --force
npx expo start -c
```

### Решение 6: Добавьте папку в исключения антивируса
Добавьте папку `C:\Users\kanazxz\Desktop\mobile_crm\node_modules` в исключения вашего антивируса.

### Решение 7: Проверьте файервол Windows
1. Откройте "Брандмауэр Windows в режиме повышенной безопасности"
2. Создайте правило для входящих подключений для node.exe
3. Разрешите подключения на порту 8081

## Если ничего не помогает

Попробуйте запустить на другом компьютере или используйте WSL (Windows Subsystem for Linux).
